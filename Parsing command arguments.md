# Parsing command arguments

## Общее

Так как команда ``grep`` имеет нетривиальный разбор ключей, то он был реализован с использованием библиотеки для
разбора аргументов командной строки.

## Рассмотренные варианты библиотек

Перед реализацией ``grep`` были изучены различные библиотеки, осуществляющие разбор аргументов командной строки.

### Apache Commons CLI

[Документация](https://commons.apache.org/proper/commons-cli/)

Плюсы:

- Можно настроить автогенерируемую справку;
- Позволяет сделать группы опций;
- С помощью групп опций возможна настройка исключающих опций.

Минусы:

- Старая, новая версия ещё в разработке;
- Не очень удобная работа с функционалом, но есть обёртки, которые облегчают (рассмотрены ниже);

### JCommander

[Документация](http://jcommander.org/)

Является обёрткой над [Apache Commons CLI](#apache-commons-cli), предоставляет возможность взаимодействовать с опциями
при помощи аннотаций, что облегает написание кода и улучшает его читабельность.

Плюсы:

- Поддерживает паттерн Команда;
- Предоставляет возможность работы с помощью аннотаций;
- Перехват исключений при несоответствии команде с возможностью вывода справки об ошибке;
- Нет фиксированного порядка параметров;
- Удобная документация.

Минусы:

- Как и в [Apache Commons CLI](#apache-commons-cli);
- Нет возможности задания нескольких параметров без имени, из-за того, что нет фиксированного порядка параметров.

### Airline

[Документация](https://rvesse.github.io/airline/)

Плюсы:

- Хорошая документация;
- Удобная работа через аннотации.

Минусы:

- Сложность в подключении к проекту.

### CLAJR

[Документация](https://clajr.sourceforge.net/)

Плюсы:

- Готовый ``.jar`` файл для работы;

Минусы:

- Старая библиотека без поддержки;
- Нет поддержки аннотаций;
- Нет гибкой настройки взаимодействия параметров.

## Заключение

После ознакомления анализа различных библиотек, предназначенных для разбора аргументов командной строки, было решено 
использовать библиотек [JCommander](#jcommander), так как она предоставляет необходимый функционал, удобный в 
использовании, а также имеет хорошую документацию.
